# User Guide - Eye Tracking System

## Getting Started

### Camera Setup
1. Position your webcam at eye level
2. Sit about 50-70 cm from the camera
3. Ensure your face is well-lit

### Launch
```bash
python src/main.py
```

## Calibration

Press `C` to start calibration. Look at each target for 2 seconds.

## Controls

| Key | Action |
|-----|--------|
| Q/ESC | Quit |
| C | Calibrate |
| K | Toggle keyboard |
| M | Toggle mouse |
| SPACE | Manual click |

## Tips

- Recalibrate when you change position
- Use good lighting
- Keep head still during calibration
